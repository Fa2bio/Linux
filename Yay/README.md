<h1>YAY</h1>

### Install Make
```xml
sudo pacman -S make
```

## Install Base-Devel
```xml
pamac install base-devel
```

### Install Git
```xml
sudo pacman -S git go
```

### Clone YAY
```xml
git clone https://aur.archlinux.org/yay.git
```

### Find YAY Folder
```xml
cd yay
```

### Build YAY
```xml
makepkg -si
```
